Difference between var, let, and const:
var can be redeclared.
let can be changed but cannot redeclare.
const cannot change value.

The Spread Operator:
The spread operator is used to spread out values from an array or object.

Difference between map(), filter(), and forEach():
map() creates new array.
filter() items select using a condition.
forEach() run code for every items in array.

Arrow Function:
An arrow function is a short way to write a function.

Template literals:
Template literals used to write strings easily using backticks.
